# Specification - Edit Chat Title Icon Color

## Problem
The "edit chat title" icon color is inconsistent with other icons in the sidebar, specifically the unpinned state of the pin icon.

## Proposed Solution
- Update the CSS to ensure the edit icon color matches the unpinned icon color.

## Requirements
- Identify the CSS class/selector for the edit chat title icon.
- Identify the color used for the unpinned pin icon.
- Apply the same color to the edit icon.
